## Overview

HRMSForge is a production-grade, enterprise-level QA automation framework built for the OrangeHRM Human Resource Management System. It demonstrates mid-to-senior SDET skills across multiple testing dimensions.

## Tech Stack

| Layer            | Tool                               |
| ---------------- | ---------------------------------- |
| UI Automation    | Playwright + TypeScript            |
| API Testing      | Playwright APIRequestContext       |
| Accessibility    | @axe-core/playwright (WCAG 2.1 AA) |
| Performance      | k6 (Grafana)                       |
| Reporting        | Allure Report                      |
| CI/CD            | GitHub Actions                     |
| Containerization | Docker                             |
